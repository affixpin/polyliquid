REPUTATION FORMULA

Each correct vote adds stake. Old votes decay.

R(t) = Σ LP_stake × 0.995 ^ (days since vote)      Ceiling = stake_day / 0.005
Each correct vote adds the LP stake amount to reputation.
Old votes decay: −0.5% per day. Half-life: 138 days.
Ceiling at $1M/day stake: $1M / 0.005 = $200M.
If a voter stops voting — reputation halves in 138 days.
Protection against abandoned accounts and dormant voters.
